# UgovorManagementApplication

To run the application, you need to have installed:
- Java 21
- docker
- docker-compose

To run the application, you need to execute the following commands:

```docker-compose up```

```gradle bootRun```

Open API documentation is available at http://localhost:8080/swagger-ui.html

To run the tests, you need to execute the following command:

```gradle test```
